Ask Hub

Ask Hub is a simple Q&A platform where users can ask questions, get answers, and interact with a community. Built with a traditional frontend and a Node.js/Express backend.

🔹 Features

Ask and answer questions

Responsive and user-friendly interface

RESTful API using Node.js & Express

Environment configuration with .env

💻 Technologies Used

Frontend: HTML, CSS, JavaScript

Backend: Node.js + Express

Environment Variables: .env

🚀 Installation

Clone the repository:

git clone https://github.com/your-username/Askhub.git
cd ask-hub


Install backend dependencies:

npm install


Create a .env file in the root directory and add:

PORT=3000
DATABASE_URL=your_database_url


Start the server:

npm start


Open the frontend (index.html) in your browser to use the app.
